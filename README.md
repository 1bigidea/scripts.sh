# scripts.sh

## Usage

1. Clone repository
2. Add entire folder to your `$PATH`
````
echo '# scripts.sh (https://github.com/dannyvankooten/scripts.sh)' >> ~/.bash_profile
echo 'export PATH="~/Scripts:$PATH"' >> ~/.bash_profile
```
3. Enjoy.


### WP Plugin Releaser
```sh
$ release-plugin $DIRECTORY $VERSION
```
